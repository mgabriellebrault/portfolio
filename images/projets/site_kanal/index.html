Concept

Comment ça marche?

L’application Kanal c’est plus qu’un guide, c’est une vraie plus-value de l’expérience de l’art contemporain dont vous en ressortirez enrichi.

Télécharger

L’application Kanal est disponible sur Play Store. Vous pouvez choisir de l’installer sur votre téléphone ou d’emprunter un appareil fourni par le centre.

S’inscrire

Créer votre profil afin de vous tenir au courant de tous les évènements du centre ainsi que des activités dont vous vous êtes inscrit.

Visiter

Profitez d’une expérience de visite qui sera aussi bien digitale que sociale. Vous en apprendrez plus sur les artistes, les œuvres, les autres mais aussi vous-même.

S’exprimer

Après votre visite, vous pourrez exprimer votre ressenti et le confronter à celui d’autres personnes.

Faq

Questions et réponses

Une nouvelle application apporte toujours son lot de questions. Afin de mieux vous guider dans ce questionnement, nous avons compilé les questions les plus fréquentes.

Quel genre d’application est Kanal ?

Kanal est une application culturelle liée à Kanal Centre Pompidou. Elle permet de vivre une expérience digitale et sociale du centre.

Est-ce que je peux utiliser Kanal gratuitement ?

Après l’achat de du ticket de visite oui, puisque le prix de l’application est comprise dans le prix de la visite si vous avez choisi l’option visite avec l’application.

Est-il possible d’utiliser Kanal sans se connecter à internet ?

Non, l’application nécessite une connexion internet. Le centre dispose d’une connexion wifi dans tout le bâtiment.

Avec quel système d’exploitation est-il possible d’utiliser cette application ?

L’application est compatible avec le système d’exploitation Android, si vous n’en avez pas, aucun souci, le centre vous prête un appareil pour la visite.

Quelles sont les langues disponibles ?

Les langues disponibles sont : le Français, le Néerlandais et l’Anglais.

Est-ce que je peux utiliser Kanal avec les mêmes données de sauvegarde sur un autre appareil ?

Oui si vous vous êtes bien inscrit en donnant votre email, il est possible d’utiliser Kanal sur un autre appareil.

Comment changer de langue ?

Suivez les instructions ci-dessous pour changer de langue.

  • Depuis le menu principal, sélectionnez Paramètres
  • Sélectionnez Général
  • Dans langue, bougez le curseur sur la langue désirée.

Comment désactiver les notifications push ?

Pour modifier les paramètres des notifications push, vous devez modifier les réglages correspondants de système d’exploitation de votre appareil.
